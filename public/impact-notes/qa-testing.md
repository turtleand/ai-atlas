# QA Testing
**Status:** Frontier

## Why this area is shifting
AI can generate test cases, write assertions from specs, and even identify flaky tests. Manual QA for straightforward user flows is being automated. But edge cases, exploratory testing, and understanding user intent still need humans.

## Timeline
- **2024-05** - AI-powered test generation tools (Codium, Diffblue) see growing adoption
- **2024-11** - GitHub Copilot adds test generation as a core feature
- **2025-04** - Companies report 50-70% reduction in manual test writing time
- **2025-10** - Visual regression testing with AI reaches production quality
- **2026-01** - QA role shifts toward test strategy, edge case discovery, and AI test review
