# Basic Prompt Engineering
**Status:** Submerged

As models get smarter, the need for elaborate prompting techniques decreases. Simple tricks like few-shot examples and chain-of-thought are being absorbed into model capabilities. The models figure out what you mean without careful prompt construction.
