# Simple Bug Fixes
**Status:** Submerged

## Why this area is automated
Straightforward bugs with clear error messages and stack traces are easy for AI to diagnose and fix. Given the error context and codebase, models can identify the root cause and generate patches for common issues like null references, off-by-one errors, and type mismatches.

## Timeline
- **2024-03** - GitHub Copilot adds "Fix this" inline suggestions for common errors
- **2024-09** - SWE-bench shows AI models resolving 30%+ of real GitHub issues autonomously
- **2025-02** - Anthropic's Claude achieves 49% on SWE-bench Verified
- **2025-06** - Cursor and Windsurf handle most lint errors and type issues automatically
- **2025-11** - Agentic tools routinely fix simple bugs from issue descriptions alone
