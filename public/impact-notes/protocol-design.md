# Protocol Design
**Status:** Safe

Protocol design remains above the waterline because the hard work is not only writing contract code. It is deciding how settlement, custody, identity, permissions, dispute paths, incentives, and regulatory constraints fit together without breaking the system.

This note records Turtleand's interpreted signals from public news sources. It is not a universal disruption score.
