# Documentation
**Status:** Frontier

## Why this area is shifting
AI is excellent at generating documentation from code: API references, README files, inline comments, and even tutorials. The gap is closing fast for technical docs. However, good documentation that tells a story, explains "why" decisions were made, and guides users through complex workflows still benefits from human authorship.

## Timeline
- **2024-01** - Mintlify, Readme.com integrate AI doc generation
- **2024-06** - GitHub Copilot generates docstrings and README sections inline
- **2025-02** - AI-generated API documentation reaches production quality for most projects
- **2025-09** - Technical writing roles shift toward editing AI drafts rather than writing from scratch
- **2026-01** - Reference documentation is largely automated; narrative docs remain human-driven
