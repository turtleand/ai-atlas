# Boilerplate/CRUD Code
**Status:** Submerged

Writing standard CRUD operations, REST endpoints, and boilerplate setup code was one of the first tasks AI coding tools could reliably handle. Models trained on millions of repos generate these patterns faster and more consistently than humans.

Developers can now leverage Anthropic's Claude Agent and OpenAI's Codex directly in Xcode to tackle complex tasks autonomously.
